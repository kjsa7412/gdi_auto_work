단계별 상세 데이터 흐름

/cache-refresh
상태 변경 없음
Convention Cache 갱신 (DB → 로컬 캐시)

입력 (DB 조회)

  • code_select.sql → sy_code_mst, sy_code_dtl
  • table_select.sql → 테이블/컬럼/PK/코멘트
  • view_select.sql → 뷰 정의
  • function_select.sql → fn_*/sp_* 함수
  • trigger_select.sql → 트리거 정의

출력 (캐시 파일)

  • system/cache/convention/code.txt
  • system/cache/convention/table.txt
  • system/cache/convention/view.txt
  • system/cache/convention/function.txt
  • system/cache/convention/trigger.txt
