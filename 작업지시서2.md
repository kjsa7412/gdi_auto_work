1. 구현 코드를 보면 다음과 같이 말도 안되는 함수로 구현된걸 알 수 있다.
이러한 원인이 발생한 이유를 말해라


 파일 전체를 올바른 프레임워크 API로 재작성 필요 항목

  ┌──────────────────────────────────┬──────────────────────────────────────────────────────────────────────┐
  │     기존 (존재하지 않는 API)     │                       수정 후 (프레임워크 API)                       │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.alert(...)              │ popup.alert.show(...)                                                │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.confirm(...)            │ popup.confirm.show(...)                                              │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.dataSearchForm(...)     │ listener.search.searchForm = webix.ui({view: 'dataSearchForm', ...}) │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.dataForm(...)           │ listener.form.mainForm = webix.ui({view: 'dataForm', ...})           │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.dataToolbar(...)        │ listener.toolbar.xxx = webix.ui({view: 'dataToolbar', ...})          │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.getSearchFormParam(...) │ listener.search.searchForm.getData()                                 │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.getFormValues(...)      │ listener.form.mainForm.getData()                                     │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.setFormValues(...)      │ listener.form.mainForm.setData(...)                                  │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ platform.setFormReadonly(...)    │ listener.form.mainForm.setReadonly(...)                              │
  ├──────────────────────────────────┼──────────────────────────────────────────────────────────────────────┤
  │ function () {} (콜백)            │ new Callback(function (result) {})                                   │
  └──────────────────────────────────┴──────────────────────────────────────────────────────────────────────┘
